Endpoints

GET /api/new

Generates a new random temporary email.

/api/new?username=optional

GET /api/inbox

Fetch all messages for a specific token.

/api/inbox?token=YOUR_TOKEN

GET /api/message

Read a specific message and auto-detect OTPs.

/api/message?id=MSG_ID&token=YOUR_TOKEN

DELETE /api/delete-all

Clears the entire inbox for a token.

/api/delete-all?token=YOUR_TOKEN
